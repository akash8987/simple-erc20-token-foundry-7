# Simple ERC-20 Token (Foundry)

This repository contains a professional-grade, secure implementation of a standard ERC-20 token using the OpenZeppelin library. It is designed to be a "plug-and-play" template for Web3 developers.

## Features
* **Standard Compliant**: Fully compatible with the ERC-20 interface.
* **Secure**: Utilizes OpenZeppelin's battle-tested `ERC20.sol` contract.
* **Modern Tooling**: Built for the Foundry development framework.

## Getting Started
1. Install Foundry: `curl -L https://foundry.paradigm.xyz | bash`
2. Install dependencies: `forge install OpenZeppelin/openzeppelin-contracts`
3. Compile: `forge build`
4. Test: `forge test`
